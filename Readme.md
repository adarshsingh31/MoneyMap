# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/adarshsingh31/MoneyMap.git
```

```bash
cd MoneyMap
```

---

## Backend

```bash
cd backend
npm install
```

Create a `.env`

```env
MONGO_URI=your_connection_string
JWT_SECRET=your_secret
PORT=5000
```

Run

```bash
npm start
```

---

## Frontend

```bash
cd frontend
npm install
```

Create a `.env`

```env
VITE_API_URL=http://localhost:5000/api
```

Run

```bash
npm run dev
```
